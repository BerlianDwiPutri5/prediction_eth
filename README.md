## Deskripsi
Project ini merupakan tugas akhir (skripsi) yang bertujuan untuk melakukan prediksi harga cryptocurrency Ethereum menggunakan metode regresi linear berganda. Studi ini difokuskan pada aset Ethereum (ETH) untuk membantu memahami pola pergerakan harga berdasarkan data yang tersedia.

## Tujuan
- Menerapkan metode regresi linear berganda dalam analisis data
- Melakukan prediksi harga cryptocurrency, khususnya Ethereum (ETH)
- Membantu investor dalam menentukan strategi berdasarkan hasil analisis data
